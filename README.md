# Cuisin'Art

Cuisin'Art est un site web de recettes de cuisine, conçu pour aider les amateurs de cuisine à découvrir de nouvelles idées de plats et à perfectionner leurs compétences culinaires.

## Fonctionnalités

- Affichage des recettes populaires
- Recherche de recettes par ingrédients, catégories, etc.
- Création d'un compte utilisateur pour sauvegarder des recettes préférées
- Ajout de nouvelles recettes par les utilisateurs

## Technologies Utilisées

- React.js : Une bibliothèque JavaScript pour la construction d'interfaces utilisateur
- Sass : Un préprocesseur CSS pour faciliter le style et la mise en page
- Vite : Un outil de construction rapide pour les applications web modernes

## Installation

1. Cloner le repository :
```

git clone https://github.com/Escanor1986/Cuisin-Art

```

2. Accéder au répertoire du projet :

```

cd cuisin-art

```

3. Installer les dépendances :

```

npm install

```

4. Démarrer le serveur de développement :

```

npm run dev

```

5. Ouvrir votre navigateur et accéder à l'URL suivante :

```

http://localhost:5173

```
